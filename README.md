## React 完整项目示例

### 运行项目 

```
yarn OR npm install
```

### 如果安装速度很慢，设置淘宝镜像源

```
yarn config set registry https://registry.npm.taobao.org
OR 
npm config set registry https://registry.npm.taobao.org
```

### 运行项目

```
yarn start OR npm run start
```

### 打包项目

```
yarn build OR npm run build
```

### 打开浏览器预览项目 

```
http://localhost:3000
```
